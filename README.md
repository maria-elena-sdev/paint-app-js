# 🎨 Paint App

A simple and interactive browser-based paint application built with vanilla JavaScript, HTML5 Canvas, and CSS.

---

## ✨ Features

- Freehand drawing on canvas
- Multiple color palette options
- Adjustable brush size
- Fill mode for coloring the entire canvas
- Save drawings as PNG images
- Responsive canvas resizing
- Quick canvas reset button

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- HTML5 Canvas API

---

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/maria-elena-sdev/paint-app-js.git
```

### Open the project

Simply open the `index.html` file in your browser.

No installation or external dependencies required.

## 🎮 Controls

| Action | Description |
|--------|-------------|
| Mouse Drag | Draw on the canvas |
| Fill Button | Fill the entire canvas with the selected color |
| Save Button | Download artwork as PNG |
| Brush Slider | Adjust brush size |
| Refresh Button | Clear and reset the canvas |
